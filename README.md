# Cypress
## Project Overview
This project contains end-to-end automation tests for the STC Child Budget web application:
https://stcchildbudget.infodev.com.np/

The platform is used for managing and tracking child-related government budget data.

## Tools & Technologies
- Cypress
- JavaScript
- Node.js

## Test Scenarios Covered
- Login functionality
- Navigation and dashboard validation
- Form submissions
- Data validation
- UI behavior and edge cases

## How to Run
1. Clone the repo
2. Run: npm install
3. Run: npx cypress open
